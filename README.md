<h1 align = "center"> 🔍 UniFind — University Lost & Found System </h1>

A full-stack web application that helps university students report and recover lost or found items on campus.

---

## 🌟 Features

- 📢 Post lost or found items
- 🖼️ Upload image of the item
- 🔍 Filter by type, category and status
- 📊 Live stats — lost, found and resolved counts
- ✅ Mark items as resolved when returned
- 🗑️ Delete posts
- 📱 Responsive and clean UI

---

## 🧱 Tech Stack

| Part       | Technology        |
|------------|-------------------|
| Frontend   | React             |
| Backend    | Go + Gin          |
| Database   | MongoDB (Local)   |
